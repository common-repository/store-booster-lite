== Description ==
This is powerful and useful addon for WooCommerce, this plugin adds some extra features to the WooCommerce which definitely will make your online life easier. This plugin is very useful tool for any ecommerce sites. It adds useful function to WooCommerce and helps to increase your revenue.


=== Features === 
Adds Product Quick View
Adds Ajax product search options
Adds Sticky bar on single product page to urge the viewer to buy products
Displays product sales counter
Adds login/signup option to your store
Easily configuration with shortcodes & menu options
Translation Ready
Lifetime Free support
Constant updates

== Installation ==
1. Unzip store-booster-lite.zip
2. Upload all the files to the /wp-content/plugins/store-booster-lite
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on Store Booster Lite option menu in Wordpress left admin menu.
